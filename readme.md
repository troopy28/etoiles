# Gamejam Espace

Le projet Unity en lui-même est dans `GameJam Espace`.

## Structure des Assets

```
Assets/
├── Art/
│   ├── Materials/
│   ├── Models/
│   └── Textures/
├── Audio/
│   ├── Music/
│   └── Sound/
├── Code/
│   ├── Scripts/
│   └── Shaders/
└── Level/
    ├── Prefabs/
    ├── Scenes/
    └── UI/
```
